💳 Nubank Landing Page Replica

Este projeto é uma reprodução fiel (pixel-perfect) da landing page do Nubank, desenvolvida com o objetivo de praticar conceitos avançados de estilização, arquitetura de componentes e layout moderno com CSS.

🚀 Tecnologias Utilizadas

React (Interface baseada em componentes)

Vite (Ambiente de desenvolvimento ultra-rápido)

CSS3 (Variáveis CSS, Flexbox e CSS Grid para layouts complexos)

Google Fonts (Integração da fonte Inter)

🎯 Desafios e Aprendizados
O maior foco deste projeto foi a fidelidade visual em relação ao design original. Durante o desenvolvimento, foram aplicadas técnicas para:

Layout Estruturado: Uso de CSS Grid para organizar a seção Hero, garantindo o equilíbrio entre o conteúdo textual e o card de captura.

Centralização e Proporção: Implementação de containers de largura máxima (max-width) para manter a harmonia visual em diferentes resoluções de tela.

Estilização Avançada: Aplicação de filtros e gradientes sobre imagens de fundo para garantir legibilidade e contraste, além de bordas arredondadas e sombras para replicar o design "limpo" da marca.

Interatividade: Criação de estados de hover e focus em inputs e botões seguindo o guia de estilo do Nubank.
